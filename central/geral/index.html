Bem-vindo ao Sistema de Busca

Para buscar um artigo, você pode:

  1. Usar o formulário acima para buscar por UID
  2. Acessar diretamente via URL: ?uid=SEU_UID&collection=SUA_COLECAO
  3. Ver os artigos mais recentes clicando em "Artigos Recentes"
  4. Se estiver logado, acessar sua página pessoal clicando em "Minha Página"

Exemplo de URL: ?uid=artigo123&collection=materiadeensaio
